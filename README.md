Research Topic
LinkedIn Organic Content Strategy for B2B SaaS
Why I chose this topic

I chose LinkedIn organic content strategy for B2B SaaS because LinkedIn is widely used by professionals and founders to share ideas, build credibility, and generate leads.

This topic helps understand how content can be used not just for visibility, but also for practical business outcomes like audience growth and customer acquisition.

How I collected the data
LinkedIn posts were collected manually
YouTube transcripts were extracted using AI tools (Claude/Codex)
I did not use APIs; transcripts were gathered using available tools and manual methods
AI tools were used mainly to organize and format the content
